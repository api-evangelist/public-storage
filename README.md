# Public Storage (public-storage)

Public Storage is an American real estate investment trust (REIT) that acquires, develops, owns, and operates self-storage facilities throughout the United States and Europe. Headquartered in Glendale, California, it is one of the largest self-storage operators in the world. This repository captures the APIs, developer tools, and machine-readable API artifacts for Public Storage.
